Привет!, это мой первый DataBase проект.Мой проект представляет собой мини-прототип сети супермаркетов Magnum.Я добавил возможность заработать Бонусы, при условии того 
что клиенты будут подключены к Magnum Club, путем совершения покупок.Бонусы составляют 1% от покупки.Бонусы могут изпользованы к частичной или полной оплаты следующих 
заказов.

Get a number 0123456789 as an UPC code of product;
